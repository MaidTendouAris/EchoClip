# EchoClip Icon

Source asset:

- `apps/echoclip/assets/brand/echoclip_icon.svg`

Design intent:

- The rounded square uses a clean off-white tile so the icon stays simple at small sizes.
- The waveform is the whole mark: no loop arrow, playback marker, or extra decoration.
- The cyan-to-blue accent bar gives the minimal waveform a recognizable EchoClip identity.

Suggested export targets:

- Android adaptive icon foreground: export from the SVG at 432 x 432 px with safe padding.
- Android launcher icon PNGs: generate from a 1024 x 1024 px source.
- Windows `.ico`: include 16, 24, 32, 48, 64, 128, and 256 px sizes.
